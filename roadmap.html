Iteration 1 Milestone Roadmap

Milestone 1

User and Problem Analysis

Define core personas, map pain points, and scope the first release to the City of Melbourne LGA.

Milestone 2

Epics and User Stories

Create epics and stories with clear acceptance criteria, then map dependencies into the planning board.

Milestone 3 - Epic 1

Landing and Navigation

Build the app shell, route structure, and clear entry points, with persistent access to crisis support.

Milestone 4 - Epic 2

First-Time Guides

Publish guide content with category filters, keyword search, inline data widgets, and bookmark support.

Milestone 5 - Epic 3

Near Me

Ship suburb/geolocation search, topic filtering, synchronized list-map results, and demographic snapshots.

Ongoing

Evaluation and Iteration 2 Prep

Run user feedback loops, maintain project documentation, and scope the next iteration's priorities.
